# Lotto Number Generator

A simple, mobile-responsive Lotto Number Generator 6/45.

## Features
- **Random Generation:** Generates 6 unique numbers (1-45).
- **Responsive Design:** Works on Mobile and Desktop.
- **Dark Mode:** System preference detection & toggle switch.
- **Animation:** CSS animations for ball reveals.

## Deployment
This project is designed to be deployed on **Cloudflare Pages**.

### Configuration for Cloudflare Pages
- **Framework Preset:** `None`
- **Build Command:** `(Empty)`
- **Build Output Directory:** `public`

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
